Move Masters OS (Fleet Flow)
Version: 1.2.0 (Lifecycle Master)
Status: Operational / Private
Overview
Move Masters OS is an execution engine designed to enforce "Operational Honesty" in logistics. It replaces the traditional CRM with a hard-coded workflow that physically prevents the movement of goods until liability and financial gates are cleared.
Core Protocols
• Origin Hard Stop: Drivers cannot load until the client signs the inventory witness record.
• Destination Hard Stop: Drivers cannot unload until the Office verifies funds.
• Chain of Custody: Tracks assets through warehouse transfers and outside carrier (MC#) handshakes.
• Revenue Recovery: Real-time commission auditing for drivers based on overage capture.
Tech Stack
• Framework: React + Vite
• Styling: Tailwind CSS
• Icons: Lucide React
Development
1. Clone the repo.
2. Run npm install to install dependencies.
3. Run npm run dev to launch the local fleet server.
